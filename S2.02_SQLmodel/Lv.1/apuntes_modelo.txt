Cómo marcar por modelo que una marca solo puede tener un proveedor?
Tabla intermedia "Marcas" con una relacion 1:N desde Proveedores - CREATE TABLE Marcas (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nombre VARCHAR(50) UNIQUE,
    proveedor_id INT,
    FOREIGN KEY (proveedor_id) REFERENCES Proveedores(id)
);
Y en gafas referencias a la marca: - CREATE TABLE Gafas (
    id INT PRIMARY KEY,
    marca_id INT,
    ...
    FOREIGN KEY (marca_id) REFERENCES Marcas(id)
);

RESUMEN: Cada marca tiene un único proveedor. Una marca puede estar asociada a muchas gafas, y nunca se repite la asociación marca-proveedor
